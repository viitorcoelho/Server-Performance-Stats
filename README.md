# 📊 Server Stats Script

Script em Bash para análise básica de performance de servidores Linux.  
Pode ser executado em qualquer ambiente, incluindo instâncias na AWS (EC2).

---

## 🚀 Funcionalidades

O script coleta e exibe:

- Uso total de CPU  
- Uso de memória (livre vs usado + porcentagem)  
- Uso de disco (livre vs usado + porcentagem)  
- Top 5 processos por uso de CPU  
- Top 5 processos por uso de memória  


## 🛠️ Tecnologias

- Bash Script  
- Comandos nativos Linux:
  - `top`
  - `free`
  - `df`
  - `ps`
  - `uptime`
  - `who`
 
## Exemplo de saída
===== SERVER PERFORMANCE STATS =====

>> CPU Usage:
CPU Used: 12.5%

>> Memory Usage:
Used: 1024MB | Free: 2048MB | Usage: 33.33%

>> Disk Usage:
Used: 8G | Free: 20G | Usage: 30%

>> Top 5 Processes by CPU:
PID   CMD   %CPU

# 📊 Server Stats Script

A Bash script for basic Linux server performance analysis.  
It can be executed in any environment, including AWS (EC2) instances.

---

## 🚀 Features

The script collects and displays:

- Total CPU usage  
- Memory usage (free vs used + percentage)  
- Disk usage (free vs used + percentage)  
- Top 5 processes by CPU usage  
- Top 5 processes by memory usage  

---

## 🛠️ Technologies

- Bash Script  
- Native Linux commands:
  - `top`
  - `free`
  - `df`
  - `ps`
  - `uptime`
  - `who`
